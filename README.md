# 📘 InsightAI – Intelligent Document Query System

InsightAI is an AI-powered document question-answering system built using FastAPI, React, LangChain, FAISS, and Hugging Face Transformers.

The application allows users to:

- Upload PDF documents
- Process and store document embeddings
- Ask questions based on uploaded content
- Retrieve intelligent answers using Retrieval-Augmented Generation (RAG)

---

# 🚀 Features

✅ Upload PDF documents  
✅ Automatic text chunking  
✅ Semantic similarity search using FAISS  
✅ RAG-based question answering  
✅ FastAPI backend  
✅ React + Vite frontend  
✅ Hugging Face Transformer integration  
✅ Clean and responsive UI
